xml_load_string

#include "sflxmll.h"
int
xml_load_string (XML_ITEM  **item,
                 const char *xmlstring,
                 Bool allow_extended)

Synopsis

Loads an XML string into a new XML tree. The XML data is not checked against a DTD. See xml load() for details.
